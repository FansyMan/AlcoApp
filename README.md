# AlcoApp
My 1st project

Моё первое приложение, вместо To-Do листа сделал немного позабавнее</br>
Можно добавить информацию о понравившемся напитке, где его купить и свою оценку</br>
Информация хранится в табличке, при нажатии - открывается полная информация</br></br>

Технологии:</br></br>

Приложение написано только на Swift, без использования сторонних библиотек</br>
Приложение сделано при помощи Storyboards и XIBs</br></br>

Приложение использует для хранения CoreData</br>
Используется UITableView для отображения списка напитков</br>
Добавлен ScrollView для отображения кнопок и текстовых полей при появлении клавиатуры</br>
Используется UICollectionView для отображения звёзд рейтинга</br>
Довольно-таки массивно сделанная анимация на первом экране, но тогда мне казалось это крутым)</br></br>

Нет архитектуры, приложение написано некрасиво, много повторений. Повторюсь, это первое моё приложение</br>
